man cd

NOM

cd - change le répertoire de travail

SYNOPSIS

cd [chemin du répertoire]

DESCRIPTION

La commande cd permet de changer le répertoire de travail du terminal pour un autre répertoire donné
/ est utilisé pour séparé les répertoire du chemin donné
./ signifie le répertoire courant
../ signifie le répertoire parent du répertoire courant

EXEMPLE

user
|
|-----dossier1
|
|-----dossier2
||
||-----dossier3

Ici la commande cd user/dossier1 place le terminal dans dossier1
Depuis dossier1, la commande cd ../dossier2/dossier3 place le terminal dans dossier3
