man cat

NOM

cat - lit le contenu d'un fichier

SYNOPSIS

cat [nomDuFichier]

DESCRIPTION

La commande cat permet d'afficher le contenu d'un fichier dans la sortie standard du terminal

EXEMPLE

Soit un fichier truc1 contenant le mot "toto" et un fichier truc2 contenant le mot "tata"

user
|
|-----dossier1
||-----truc1
||-----truc2

Depuis dossier1, la commande cat truc1 affichera:
toto
Depuis dossier1, la commande cat truc2 affichera:
tata
