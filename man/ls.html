man ls

NOM

ls - liste les fichiers et répertoires du répertoire courant

SYNOPSIS

ls

DESCRIPTION

La commande ls permet d'afficher les différents fichiers et répertoire se trouvant dans le répertoire courant

EXEMPLE

user
|
|-----dossier1
|
|-----dossier2
||
||-----fichier1
||-----fichier2

Depuis le répertoire user, la commande ls affichera:
dossier1
dossier2

Depuis dossier2, la commande ls affichera:
fichier1
fichier2
