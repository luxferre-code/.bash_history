man mkdir

NOM

mkdir - créer un répertoire

SYNOPSIS

mkdir [nomDuRépertoire]

DESCRIPTION

La commande mkdir créer un répertoire dans le répertoire courant

EXEMPLE

user
|
|-----dossier1
|

Depuis le répertoire user, la commande mkdir dossier2 créer un répertoire dossier2 dans le répertoire user:

user
|
|-----dossier1
|
|-----dossier2
|
